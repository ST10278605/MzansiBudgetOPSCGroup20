A personal budget management tool designed specifically for young professionals and students in South Africa.

About The App

MzansiBudget is a simple mobile personal budget management application that helps South African users track their expenses, manage budgets, and gain insights into their spending habits. 
The app features local expense categories like Transport, Airtime/Data, Groceries, Entertainment, and Savings.

Features On The App

User Authentication: Secure login and account registration 

Expense Tracking: Add expenses with amount, date, time, description, and category

Category Management: Create, edit, and delete custom expense categories

Budget Goals: User is able to set minimum and maximum monthly spending goals

Reports and Analytics: Users is able to view spending breakdown by category with interactive pie charts

Date Range Filtering: User is able to filter expenses by this month, last month, last 3 months, or this year

Dashboard Display: See total spent, remaining budget, and recent expenses at the end.
